# Lumina

Lumina 是 LeavesMC 开发的一个 Folia 分支,旨在遵循原版生电特性的同时 Folia 的功能

:::tip

目前 Lumina 处于早期开发阶段,暂时不完善,推荐去使用 Luminol

:::
