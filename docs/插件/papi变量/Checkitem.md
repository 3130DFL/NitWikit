## 我该如何安装和更新 CheckItem 拓展
**在游戏中或在管理员后台输入以下指令**
```
/papi ecloud download CheckItem     //下载 CheckItem (请确保你正处于联网状态下)
/papi ecloud update all     //更新你现有的所有拓展(包括CheckItem)
/papi reload     //重新加载插件确保 CheckItem 下载或更新完成
```
在游戏中输入时请确保你有管理员(OP)权限或者Papi的相关权限.
在管理员后台中输入时记得删除斜杆--> '/'
